Welcome to Indeed Web Scrapper’s documentation!¶

This package can be used to scrape job postings from Indeed.com. Moreover, a cli user interface is provided to enter query parameters to search for a job, and the results are displayed and can be exported to a .csv file. This package uses beautifulsoup4 4.9.1 and requests 2.24.0. In addition, I recommend using Python 3.6+.

class url_utilities.urlMaker(**params)¶

Turns a client query into a valid URL

A client provides a dictionary of query parameters. An object of this class performs the necessary parsing to generate a URL compatible with Indeed.com.

job_title¶

A string to describe the job to search for.

location¶

A string of a city, or a province (Default is unspecified).

radius¶

An int minimum radius of search in km (Default is unspecified).

job_type¶

A string specifying whether the job is full-time, part-time, etc. (Default is unspecified). Accepts one of {fulltime, parttime, internship}.

build_url()¶

Builds a URL using the existing urlMaker attributes

Changes the url so that it directs to the previous page
